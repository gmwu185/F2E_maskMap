# maskMap
- [成品連結](https://gmwu185.github.io/F2E_maskMap/index.html)
## 技術內容
- OpenStreetMap (OSM - 開源圖資服務)。
- Leaflet (圖資 JavaScript 框架)。
- 手刻收合選單配合 Map 自動調整畫面寬度佔位。
- 手刻 loading 畫面待接收完成 AJAX 傳回資料。
- 判斷取得地理位置功能做預設定位點顯示畫面。
